Sistema Integral para la Gestión de Mantenimiento (SIGM)
Este proyecto nace de la necesidad de modernizar y digitalizar el control operativo del mantenimiento en maquinaria pesada. El sistema permite gestionar de manera eficiente las intervenciones técnicas, el estado de la flota y la disponibilidad de recursos, eliminando la dependencia de registros manuales propensos a errores.
